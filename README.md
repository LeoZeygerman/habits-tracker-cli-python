Habit Tracker CLI

Консольное приложение для отслеживания привычек на Python.

Проект позволяет создавать привычки, отмечать их выполнение и отслеживать серии выполнений (streaks). Все данные сохраняются между запусками программы.

Возможности

* Создание новых привычек
* Просмотр списка привычек
* Удаление привычек
* Отметка привычки как выполненной
* Подсчет текущей серии выполнений (streak)
* Сохранение данных в файл
* Автоматическая работа с датами

Технологии

* Python 3
* Object-Oriented Programming (OOP)
* JSON для хранения данных
* Модуль datetime

Как работает приложение

Каждая привычка хранит:

* название;
* дату последнего выполнения;
* текущий стрик.

При отметке выполнения приложение:

1. Проверяет текущую дату.
2. Сравнивает её с датой последнего выполнения.
3. Увеличивает стрик при последовательном выполнении.
4. Сбрасывает стрик при пропуске дней.
5. Сохраняет изменения в файл.

Запуск

Клонируйте репозиторий:

git clone https://github.com/LeoZeygerman/habits-tracker-cli-python.git

Перейдите в папку проекта:

cd habits-tracker-cli-python

Запустите программу:

python main.py
